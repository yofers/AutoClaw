Workflow

首次使用

快速上手
1. 安装 OpenClaw

先确认安装状态，再执行安装或更新动作，保证本地 CLI 和 Gateway 可用。

2. 运行初始化向导

保留默认值也可以，目标是尽快生成本地可工作的配置并拿到 Dashboard。

3. 用运维与配置继续维护

通过 Gateway 状态、执行输出和配置编辑器做后续调整；更复杂的流程仍可回到 CLI。

需要更完整的流程时，仍可在终端使用 openclaw onboard 或 openclaw configure。
