添加方式

点击右上角“添加频道”后，在弹出框里选择频道类型并填写配置，确认后会直接保存。

新增和编辑都通过弹出框完成。现有频道会显示在左侧列表中，点“编辑”可重新打开配置。

添加频道

先选择频道类型，再填写配置。

Telegram

Bot Token、allowFrom 与群组触发规则。

飞书

账号、App 凭证与私聊 / 群聊规则。

Slack

支持 Socket Mode 与 HTTP Events，两种模式共用访问规则。

Discord

Bot Token、DM allowFrom 与 Guild allowlist。

WhatsApp

QR 登录在运行期完成，这里主要配置 account 与访问规则。

Signal

signal-cli 账号、CLI 路径与私聊 / 群聊 allowlist。

社区目录

官方当前更接近“社区插件目录”，不是独立商城；可直接浏览收录插件和安装命令。

插件系统 Mattermost 示例

常用命令

openclaw plugins list
openclaw plugins install @openclaw/mattermost
openclaw plugins install @icesword760/openclaw-wechat
openclaw plugins remove <npm-spec>

适合后续接成真正的“安装 / 卸载 / 更新扩展”页面。
